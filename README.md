# Craig Morrison's ESLint Config

## Instructions

### Install

```shell
npm install @craigmorrison/eslint-config
```

### Configure with ESLint

#### Base

```json
{
  "extends": "@craigmorrison/eslint-config"
}
```

#### React

```json
{
  "extends": [
    "@craigmorrison/eslint-config",
    "@craigmorrison/eslint-config/react"
  ]
}
```
